
# MuseEEGHeadset-Scrubber

This repository contains scripts and notebooks for processing and cleaning data from the Muse EEG Headset, producing [Tidy Data](https://www.wikiwand.com/en/Tidy_data).

## Files

1. **DataScrub.py**: A Python script for data processing.
2. **DataScrub.ipynb**: A Jupyter notebook that provides a detailed walkthrough of the data scrubbing process.

## Usage

To use the Python script, run:
```
python DataScrub.py
```

For the Jupyter notebook, ensure you have Jupyter installed and run:
```
jupyter notebook DataScrub.ipynb
```
